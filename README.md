### Development Setup

For front-end asset management, we are using grunt.
This requires [node](http://nodejs.org/download/) and npm.

	# install grunt plugins
	npm install
	
	# watch for changes, re-build, and livereload
	grunt

	# build production-ready assets
	grunt build
